│  │                                                                          │   │

│  ┌─────────────────────────────────────────────────────────────────────────┐   │
│  │ Key Pattern: lock:{resource}                                            │   │

│  │                                                                          │   │

│  │   community_count   -> 社区数量                                          │   │
│  │   last_updated      -> 最后更新时间                                      │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
