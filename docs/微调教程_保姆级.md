<a id="step-4"></a>
## Step 4：用 GPT 批量生成高质量中文天文 QA

> 这一步非常关键！自己生成的高质量数据是微调效果的关键。

创建文件 `~/astro-finetune/generate_qa.py`，内容如下：

```python
"""
用 GPT 批量生成天文问答数据。
需要 OpenAI API Key。如果没有，可以用免费的 deepseek API 替代。
"""
import json
import time
from openai import OpenAI

# ============ 配置区（只需改这里）============
# 方式1：OpenAI
# API_KEY = "sk-你的API密钥"
# BASE_URL = "https://api.openai.com/v1"
# MODEL = "gpt-4o-mini"

# 方式2：DeepSeek（便宜，效果也不错）
API_KEY = "你的DeepSeek API密钥"  # 去 https://platform.deepseek.com 注册获取
BASE_URL = "https://api.deepseek.com/v1"
MODEL = "deepseek-chat"
# =============================================

client = OpenAI(api_key=API_KEY, base_url=BASE_URL)

TOPICS = [
    "太阳系的行星及其特征",
    "恒星的生命周期（主序星、红巨星、白矮星、中子星、黑洞）",
    "银河系的结构和组成",
    "星系的分类（椭圆星系、螺旋星系、不规则星系）",
    "大爆炸理论和宇宙的起源",
    "暗物质和暗能量",
    "系外行星的发现方法（凌日法、视向速度法）",
    "天文望远镜的类型和原理",
    "宇宙微波背景辐射",
    "引力波的发现和意义",
    "超新星爆发的机制",
    "黑洞的性质（事件视界、霍金辐射、喷流）",
    "月球的起源和特征",
    "火星探测的历史和发现",
    "木星及其卫星系统（伽利略卫星）",
    "土星环的成因和结构",
    "彗星和流星雨",
    "太阳的结构（日冕、色球层、光球层）",
    "太阳活动（太阳黑子、太阳耀斑、日冕物质抛射）",
    "开普勒三大定律",
    "广义相对论与天体物理",
    "中子星和脉冲星",
    "星云的类型（发射星云、反射星云、暗星云、行星状星云）",
    "哈勃定律和宇宙膨胀",
    "詹姆斯·韦伯太空望远镜的科学目标",
    "中国天文学成就（FAST、嫦娥工程、天问一号）",
    "天文单位和距离测量方法（视差法、标准烛光）",
    "HR图（赫罗图）的意义",
    "潮汐锁定现象",
    "拉格朗日点的应用",
]

SYSTEM_PROMPT = """你是一位专业的天文学科普专家，擅长用通俗易懂的中文解释复杂的天文概念。
请根据给定的主题，生成5个高质量的中文问答对。

要求：
1. 问题要自然，像真人会问的那样
2. 回答要详细、准确、结构清晰（200-500字）
3. 使用Markdown格式（标题、列表、加粗等）
4. 涉及具体数据时要准确（如天体质量、距离等）
5. 难度从入门到进阶递增

输出格式为JSON数组，每个元素包含 question 和 answer 字段。"""


def generate_qa_batch(topic):
    """为一个主题生成5个QA对"""
    try:
        response = client.chat.completions.create(
            model=MODEL,
            messages=[
                {"role": "system", "content": SYSTEM_PROMPT},
                {"role": "user", "content": f"主题：{topic}\n\n请生成5个关于这个主题的中文天文问答对。"}
            ],
            temperature=0.7,
            max_tokens=4000,
            response_format={"type": "json_object"}
        )
        content = response.choices[0].message.content
        data = json.loads(content)
        # 可能返回 {"qa_pairs": [...]} 或直接 [...]
        if isinstance(data, dict):
            for key in data:
                if isinstance(data[key], list):
                    return data[key]
        return data if isinstance(data, list) else []
    except Exception as e:
        print(f"  Error for topic '{topic}': {e}")
        return []


def main():
    all_qa = []
    for i, topic in enumerate(TOPICS):
        print(f"[{i+1}/{len(TOPICS)}] Generating QA for: {topic}")
        qa_pairs = generate_qa_batch(topic)
        all_qa.extend(qa_pairs)
        print(f"  Got {len(qa_pairs)} pairs, total: {len(all_qa)}")
        time.sleep(1)  # 避免API限流

    # 转为 ShareGPT 格式
    sharegpt_data = []
    for qa in all_qa:
        q = qa.get("question", qa.get("q", ""))
        a = qa.get("answer", qa.get("a", ""))
        if q and a:
            sharegpt_data.append({
                "conversations": [
                    {"from": "human", "value": q},
                    {"from": "gpt", "value": a}
                ]
            })

    output_path = "datasets/generated_astro_qa.json"
    with open(output_path, "w", encoding="utf-8") as f:
        json.dump(sharegpt_data, f, ensure_ascii=False, indent=2)

    print(f"\nDone! Generated {len(sharegpt_data)} QA pairs")
    print(f"Saved to: {output_path}")


if __name__ == "__main__":
    main()
```

运行：
```bash
cd ~/astro-finetune
python generate_qa.py
# 大约需要 5-10 分钟，生成 ~150 条高质量 QA
# 如果需要更多，增加 TOPICS 列表中的主题
```

## Step 5：将所有数据转为 LLaMA Factory 训练格式

创建文件 `~/astro-finetune/prepare_data.py`：

```python
"""将所有数据集合并转为 LLaMA Factory 的 ShareGPT 格式。"""
import json
import os
import csv

output_dir = "LLaMAFactory/data"
os.makedirs(output_dir, exist_ok=True)

all_data = []

# ── 1. 加载 GPT 生成的 QA ──
gpt_path = "datasets/generated_astro_qa.json"
if os.path.exists(gpt_path):
    with open(gpt_path, "r", encoding="utf-8") as f:
        gpt_data = json.load(f)
    all_data.extend(gpt_data)
    print(f"Loaded {len(gpt_data)} GPT-generated QA pairs")

# ── 2. 加载 Astro-QA 数据集 ──
astro_qa_dir = "datasets/raw/Astro-QA/data/ZH"
if os.path.exists(astro_qa_dir):
    count = 0
    for subdir in os.listdir(astro_qa_dir):
        subdir_path = os.path.join(astro_qa_dir, subdir)
        if not os.path.isdir(subdir_path):
            continue
        for fname in os.listdir(subdir_path):
            if not fname.endswith(".csv"):
                continue
            fpath = os.path.join(subdir_path, fname)
            try:
                with open(fpath, "r", encoding="utf-8") as f:
                    reader = csv.DictReader(f)
                    for row in reader:
                        q = row.get("Question", "").strip()
                        # 尝试获取答案
                        a = ""
                        for key in ["Answer", "Reference_Answer", "answer"]:
                            if key in row and row[key].strip():
                                a = row[key].strip()
                                break
                        if q and a:
                            all_data.append({
                                "conversations": [
                                    {"from": "human", "value": q},
                                    {"from": "gpt", "value": a}
                                ]
                            })
                            count += 1
            except Exception as e:
                pass
    print(f"Loaded {count} Astro-QA pairs")

# ── 3. 加载天文百科数据 ──
baike_path = "datasets/raw/baike_astronomy.json"
if os.path.exists(baike_path):
    count = 0
    with open(baike_path, "r", encoding="utf-8") as f:
        for line in f:
            try:
                item = json.loads(line)
                title = item.get("title", "").strip()
                content = item.get("content", "").strip()
                if title and content and len(content) > 50:
                    q = f"请详细解释什么是{title}？"
                    a = content[:2000]  # 截断过长内容
                    all_data.append({
                        "conversations": [
                            {"from": "human", "value": q},
                            {"from": "gpt", "value": a}
                        ]
                    })
                    count += 1
            except:
                pass
    print(f"Loaded {count} Baike entries")

# ── 4. 保存最终训练数据 ──
output_path = os.path.join(output_dir, "astro_train.json")
with open(output_path, "w", encoding="utf-8") as f:
    json.dump(all_data, f, ensure_ascii=False, indent=2)

print(f"\nTotal training samples: {len(all_data)}")
print(f"Saved to: {output_path}")
```

运行：
```bash
cd ~/astro-finetune
python prepare_data.py
```

bf16: true

def score_answer(answer, keywords):
    """根据关键词计算得分（0-100）"""
    if not answer:
        return 0
    found = sum(1 for kw in keywords if kw in answer)
    return min(100, int(found / len(keywords) * 100))


def evaluate_model(model, tokenizer, label="Model"):
    """评估模型"""
    scores = []
    for i, (q, kws) in enumerate(zip(EVAL_QUESTIONS, KEYWORDS)):
        inputs = tokenizer(f"<|im_start|>user\n{q}<|im_end|>\n<|im_start|>assistant\n",
                          return_tensors="pt").to(model.device)
        outputs = model.generate(**inputs, max_new_tokens=500, temperature=0.3)
        answer = tokenizer.decode(outputs[0][inputs.input_ids.shape[1]:], skip_special_tokens=True)
        score = score_answer(answer, kws)
        scores.append(score)
        if i < 5:
            print(f"  Q: {q}")
            print(f"  A: {answer[:200]}...")
            print(f"  Score: {score}")
            print()
    avg = sum(scores) / len(scores)
    print(f"\n{label} Average Score: {avg:.1f}/100")
    return avg, scores

## Step 11：集成到星智穹图项目

### 11.1 方式一：使用 LoRA 权重（推荐，节省空间）

将 `saves/astro-qwen2vl-lora/` 文件夹拷贝到你的项目 `backend/models/` 下。

修改 `backend/models/custom_model.py`，添加模型加载代码：

```python
import torch
from transformers import Qwen2VLForConditionalGeneration, AutoProcessor
from peft import PeftModel

class AstroVLModel:
    def __init__(self, base_model_path, lora_path):
        self.processor = AutoProcessor.from_pretrained(base_model_path)
        base_model = Qwen2VLForConditionalGeneration.from_pretrained(
            base_model_path,
            torch_dtype=torch.bfloat16,
            device_map="auto"
        )
        self.model = PeftModel.from_pretrained(base_model, lora_path)
        self.model.eval()

    def chat(self, question, image=None):
        if image:
            messages = [
                {"role": "user", "content": [
                    {"type": "image", "image": image},
                    {"type": "text", "text": question}
                ]}
            ]
        else:
            messages = [
                {"role": "user", "content": question}
            ]

        text = self.processor.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)
        inputs = self.processor(text=[text], images=[image] if image else None,
                               return_tensors="pt").to(self.model.device)

        with torch.no_grad():
            output_ids = self.model.generate(**inputs, max_new_tokens=1024)

        output = self.processor.batch_decode(
            output_ids[:, inputs.input_ids.shape[1]:],
            skip_special_tokens=True
        )[0]
        return output
```

## 常见问题

### Q1: 显存不够怎么办？
使用 QLoRA（4bit量化）：在 `train_config.yaml` 中添加：
```yaml
quantization_bit: 4
quantization_method: bitsandbytes
```
这样只需 **12GB 显存**。

### Q2: 训练 Loss 不下降？
- 检查数据格式是否正确
- 降低学习率到 `5.0e-5`
- 增加 `warmup_ratio` 到 `0.2`

### Q3: 模型回答重复？
- 在生成时设置 `repetition_penalty: 1.2`
- 设置 `temperature: 0.7`

### Q4: 如何用更少的数据？
即使只有 200 条高质量 QA，也能看到明显效果。关键是数据质量！
